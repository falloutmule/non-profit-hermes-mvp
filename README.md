# Non-Profit Hermes MVP

Board-visible website and operating scaffold for the Non-Profit Hermes MVP.

This repo is intentionally simple:

- `index.md` shows the public home page
- `today.md`, `current-needs.md`, `calendar.md`, `volunteer.md`, `donations.md`, `reports.md`, and `board-log.md` are the board-facing pages
- `data/` holds approved-safe JSON consumed by the pages

No raw private intake should be committed here.
